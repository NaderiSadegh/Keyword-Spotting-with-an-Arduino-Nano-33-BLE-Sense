Load audio dataset from the specified path.

This function loads an audio dataset from the given path. If the dataset is not present, it is downloaded from a specified URL. The dataset is then split into training and validation sets.

Parameters
datasetPathThe path to the audio dataset.
batchSizeThe batch size for the datasets (default: 64).
validationSplitThe fraction of the dataset to allocate to the validation set (default: 0.2).
seedThe seed value for random operations (default: 0).
outputSequenceLengthThe output sequence length for the audio data (default: 16000).
Returns
A tuple containing training and validation datasets.
Exceptions
errorHandler.errorLoadDatasetIf an error occurs while loading the dataset.
