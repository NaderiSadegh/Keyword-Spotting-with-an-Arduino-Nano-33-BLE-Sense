Introduction

This project consists of a simple calculator module (calc) and its corresponding unit tests (testCalc). The calculator module provides a basic arithmetic operation, addition. The unit tests in testCalc.py ensure the correctness of the calculator function.

Modules

  • calc: Simple calculator module with a basic arithmetic operation.
  • testCalc: Unit tests for the function in the calc module.

Calculator Module (calc.py)

The calc module provides the following function:

  • calc.addition: Calculate the addition of two numbers.
    • Example: calc.addition(5, 5) returns 10.

Unit Test Module (testCalc.py)

The testCalc module contains the following unit tests:

  • TestCalc.test_addition: Test the addition function in calc module.
    • Test cases:
      • Positive integers (5, 5): Expected result is 10.
      • Mixed integers (-1, 1): Expected result is 0.
